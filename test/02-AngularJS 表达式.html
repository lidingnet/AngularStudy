AngularJS 使用 表达式 把数据绑定到 HTML。

AngularJS 表达式写在双大括号内：{{ expression }}。
AngularJS 表达式把数据绑定到 HTML，这与 ng-bind 指令有异曲同工之妙。
AngularJS 将在表达式书写的位置"输出"数据。
AngularJS 表达式 很像 JavaScript 表达式：它们可以包含文字、运算符和变量。

实例 5+5={{ 5 + 5 }} 或 {{ firstName + " " + lastName }}
5+5= {{5+5}}

AngularJS 数字

AngularJS 数字就像 JavaScript 数字：
实例：<div ng-init="quantity=3; cost=8">总价={{quantity * cost}}</div>
总价={{quantity * cost}}

AngularJS 字符串

AngularJS 实例: <p>姓名： {{ firstname + " " + lastname }}</p> <p>姓名： <span ng-bind="firstname + ' ' + lastname"></span></p>

姓名：{{firstname + " " + lastname}}

AngularJS 对象就像 JavaScript 对象：

<div ng-app="" ng-init="person={firstName:'John',lastName:'Doe'}">

姓为

ng-init="points=[1,15,19,2,40]"
